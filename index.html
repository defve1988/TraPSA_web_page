TraPSA (Trajectory-based Potential Source Apportionment) project is developed as a free and open-source toolkit for air pollution potential locations apportionment analysis based on trajectory ensemble receptor models. TraPSA software is a graphical and powerful tool for helping air quality community using massive monitoring data and trajectory data for pollutant potential source location apportionment. TraPSA provides monitoring site database management, smart trajectory generating and handling process, pollutant temporal pattern analysis, most of existing trajectory ensemble receptor models, and sufficient GIS edit function.

TraPSA using state-of-art trajectory model, HYSPLIT, which developed by NOAA, but it can function well without HYSPLIT software suite on the computer because an unregistered version of the HYSPLIT_4 trajectory model executable are included in TraPSA. However, it is strongly recommended that users should correctly register and install HYSPLIT software in order to generate the trajectory with the latest HYSPLT version. TraPSA is developed at the center for Air Resource Engineering & Science (CARES), Clarkson Unversity.

TraPSA provides:
1. User-friendly and open-source resarch and study platform
2. Monitoring site database management
3. Smart trajectory generating and handling process
4. Pollutant temporal pattern analysis
5. Most of existing trajectory ensemble receptor models                     
6. Sufficient GIS function and extensibility to other GIS software
7. Related MATLAB package for advanced users
