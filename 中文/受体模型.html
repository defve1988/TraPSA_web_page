轨迹追踪受体模型:
Conditional Probability Function (CPF)
CPF
计算CPF模型。

句法
cpf = CPF (conc,direction,speed,c,bin,discard,plot)

描述
cpf = CPF (conc,direction,speed,c,bin,discard,plot) 返回向量cpf。[conc,direction,speed]分别为浓度、风向和风速，大小都为N×1。c为CPF模型的阈值；bin为风向的数量；discard为模型采用的最低风速，设discard等于5，那么speed小于5的数据将不会被列入CPF模型中。如果plot等于1，将返回CPF结果图。

Concentration Field Analysis (CFA)
CFA
计算CFA模型。

句法
cfa = CFA (conc,trajectory,grid,plot)

描述
cfa = CFA (conc,trajectory,grid,plot) 返回CFA模型的计算结果栅格矩阵cfa（纬度×经度）。conc为污染物浓度矩阵，大小为N×S，S为监测站点数目；trajectory为污染物浓度对应的轨迹追踪轨迹点的矩阵，大小为N×S×T，T为轨迹追踪的时间；grid为栅格大小，单位为°。如果plot等于1，CFA模型结果将会投影到世界地图上。
Concentration Weighted Trajectory (CWT)
CWT
计算CWT模型。

句法
cwt = CWT (conc,trajectory,grid,plot)

描述
cwt= CWT (conc,trajectory,grid,plot) 返回CWT模型的计算结果栅格矩阵cwt（纬度×经度）。conc为污染物浓度矩阵，大小为N×S，S为监测站点数目；trajectory为污染物浓度对应的轨迹追踪轨迹点的矩阵，大小为N×S×T，T为轨迹追踪的时间；grid为栅格大小，单位为°。如果plot等于1，CWT模型结果将会投影到世界地图上。
Residence Time Weighted Concentration (RTWC)
RTWC
计算RTWC模型。

句法
rtwc = RTWC (conc,trajectory,grid,plot,iteration)

描述
rtwc= RTWC (conc,trajectory,grid,plot) 返回RTWC模型的计算结果栅格矩阵rtwc（纬度×经度）。conc为污染物浓度矩阵，大小为N×S，S为监测站点数目；trajectory为污染物浓度对应的轨迹追踪轨迹点的矩阵，大小为N×S×T，T为轨迹追踪的时间；grid为栅格大小，单位为°；iteration为迭代边界，设iteration等于0.5，如果两次迭代之间的平均差异小于0.5%，那么迭代就会终止。如果plot等于1，RTWC模型结果将会投影到世界地图上。
Potential Source Contribution Function (PSCF)
PSCF
计算PSCF模型。

句法
pscf = PSCF (conc,trajectory,grid,plot,c)

描述

pscf = PSCF (conc,trajectory,grid,plot,c) 返回PSCF模型的计算结果栅格矩阵pscf（纬度×经度）。conc为污染物浓度矩阵，大小为N×S，S为监测站点数目；trajectory为污染物浓度对应的轨迹追踪轨迹点的矩阵，大小为N×S×T，T为轨迹追踪的时间；grid为栅格大小，单位为°；c为PSCF模型的阈值。如果plot等于1，PSCF模型结果将会投影到世界地图上。如果S大于1，那么函数将会自动采用多站PSCF模型。
