轨迹追踪（Trajectory）：
轨迹追踪指对一个理想质点在空间中运动轨迹的追踪，其方向可以是正向的也可以是逆向的。

轨迹点（Endpoints）：
 轨迹点指在一给定时间点，理想质点在空间运动轨迹的位置。

空气轨迹追踪模型（Air Parcel Trajectory Models）：
空气轨迹追踪模型是计算无限小空气团（Air Parcel）的运动轨迹的模型。常见的空气轨迹追踪模型包括美国的HYSPLIT，欧洲的FLEXTRA和日本的METEX。这些模型通常包含理论计算和实验模型，并利用相关气象数据，其结果可以为理解空气污染、有害污染物扩散和气象事件等复杂的大气过程提供有力依据。

HYSPLIT模型：
 HYSPLIT模型是美国海洋及大气层管理署开发的一种空气轨迹追踪模型。HYSPLIT模型提供了完整的空气团轨迹计算和污染物复合迁移计算（可以包括扩散、和沉降过程），被广泛运用在大气科学研究，其中也包括空气污染源解析的研究。因此，TraPSA使用最新的HYSPLIT模型HYSPLIT_4计算空气轨迹追踪。

受体模型（Receptor Modeling）：
受体模型指应用统计方法定量鉴别污染源对受体污染物浓度的贡献。受体模型结合污染物迁移模型（例如空气轨迹追踪模型）之后可以提供更多空气污染源-受体关系信息，进一步提高对污染物的源解析和大气过程的认识，从而制定更合理、高效的空气质量管理政策。

轨迹追踪受体模型（Trajectory Ensemble Receptor Model）：
轨迹追踪受体模型通过综合分析受体污染物浓度数据和相应的空气轨迹追踪数据，可以得到空气污染源的可能地理位置。目前多种不同类型的轨迹追踪受体模型被应用到研究中，例如Conditional Probability Function (CPF)，Concentration Field Analysis (CFA)，Concentration Weighted Trajectory (CWT)，Potential Source Contribution Function (PSCF)，Residence Time Weighted Concentration (RTWC)，Simplified Quantitative Transport Bias Analysis (SQTBA)。
