TraPSA software is a graphical Windows application, and no command instructions are included for data import, process and export. Therefore, programming skill is unnecessary for users. However, basic knowledge of trajectory and trajectory ensemble receptor model is required to correctly understand and interpret analysis results.

Basic entries of TraPSA project, user's guide, examples of TraPSA applications and software updates will be introduced in this part for a new user to quickly get strated.
