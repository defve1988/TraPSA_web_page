Other Function:
Wind Rose
windrose
Plot wind rose figure.

Syntax
windrose (direction,speed)

Description
windrose (direction,speed) returns a wind rose figure of [direction,speed] (N by 1 vectors).

Pollutant Pattern
plotpattern
Plot pollutant temporal patterns.

Syntax
plotpattern (date,conc,plot)

Description
plotpattern (date,conc,plot) returns a temporal pattern figure of [date,conc] (N by 1 vectors). date should be convert into numeric format using datenum. plot could be "hourly", "daily", "weekly", "monthly", "seasonal", or "all" for plotting different trmporal pattern figures.

Event Analysis
event
Find out the highest or lowest events in time series.

Syntax
events = event (date,conc,high,number,duration,plot)

Description

events = event (date,conc,high,number,duration,plot) returns a number by 2 matrix events from [date,conc] (N by 1 vectors), which contains event peak date and event peak concentration. If high equals 1, the highest events will be returned, otherwise, the lowest events will be returned. number stands for the return event number; duration is a pre-setted event duration time; and figure will be plotted if plot equals 1.
