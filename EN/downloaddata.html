Map Data:

ESRI shapefiles is required to display map in TraPSA. Two GIS data formats, ESRI shapefile (‘.shp’, ‘.dbf’ and ‘.shx’) and GeoTIFF file (‘.tif’) are available for TraPSA. Shapefiles of related regions should be download before analyzing. Note that it will take longer time to display more specified shapefiles.

Country administrative boundaries shapefile could be downloaded at this page.

Meteorological Data:

Most common meteorological data for HYSPLIT is Eta Data Assimilation System (EDAS) and North American Regional Reanalysis (NARR) for North American locations, Global Data Assimilation System (GDAS) and NCEP/NCAR Reanalysis Archive for global scale. ​These meteorological data could be download:

EDAS: ftp://arlftp.arlhq.noaa.gov/archives/edas40/
NARR: ftp://arlftp.arlhq.noaa.gov/narr/
GDAS: ftp://arlftp.arlhq.noaa.gov/archives/gdas1/
NCEP/NCAR Reanalysis: ftp://arlftp.arlhq.noaa.gov/archives/reanalysis/​
Other meteorological data: http://nomads.ncdc.noaa.gov/data.php ​

FileZilla, a free FTP downloader, is recommanded for meteorological data downloading.
